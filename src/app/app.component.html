Component projection with NgTemplateOutlet

Always prefer NgTemplateOutlet over ng-content when you need to project components. If you use ng-content for this :

  • it breaks lifecycle hooks
  • It can cause memory leaks

NgTemplateOutlet give you the opportunity :

  • to share logic between components
  • to maintain a separation of concerns
  • to access to the child component variable from the parent
